PROJETO: Make Hits

DESCRIÇÃO:
Este projeto implementa as quatro operações básicas de CRUD (Criar, Ler, Atualizar, Deletar) para gerenciar uma coleção de playlists em memória (usando um array JavaScript como um falso banco de dados). O código está no arquivo index.js.

FUNCIONALIDADES:
- Criar playlists e músicas
- Listar playlists e músicas
- Atualizar playlists e músicas
- Deletar playlists e músicas
- Listar músicas por gênero
- Listar músicas favoritas
- Gerar playlist automática com base em gênero ou favoritas

COMO EXECUTAR OS TESTES:
- Certifique-se de ter o Node.js instalado.
- Abra o terminal na pasta do projeto.
- Digite: node index.js
