PROJETO: CRUD de Playlists em JavaScript

DESCRIÇÃO:
Este projeto implementa as quatro operações básicas de CRUD (Criar, Ler, Atualizar, Deletar) para gerenciar uma coleção de playlists em memória (usando um array JavaScript como um falso banco de dados). O código está no arquivo 'playlistController.js'.

=== REFERÊNCIAS DE ESTUDO ===

Para entender o código e os métodos de array usados, assista aos seguintes tutoriais:

1. ARRAYS E .PUSH() (Criação de Playlists):
   - Tópico: Básico de Arrays, .push() e .length
   - Link: https://www.youtube.com/watch?v=XisY6rpfbiw

2. MÉTODO .FIND() (Atualização de Nome):
   - Tópico: Como buscar um item específico em um Array
   - Link: https://www.youtube.com/watch?v=szXPNFjVKzI

3. MÉTODO .FILTER() (Deleção de Playlists):
   - Tópico: Como criar um novo Array excluindo um ou mais itens
   - Link: https://m.youtube.com/watch?v=52Gmi4rRtZU

4. REVISÃO GERAL DOS MÉTODOS AVANÇADOS:
   - Tópico: forEach, map, find, filter e reduce em uma só aula
   - Link: https://www.youtube.com/watch?v=k3Yj7iEiCuU

=== COMO EXECUTAR OS TESTES ===
1. Certifique-se de ter o Node.js instalado.
2. Abra o terminal na pasta do projeto.
3. Digite: node playlistController.js
